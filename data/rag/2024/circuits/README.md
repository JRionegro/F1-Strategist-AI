# 2024 Circuit Documents

This folder contains circuit-specific documents for the 2024 season.

## Structure

Each circuit has its own subfolder with:

- `strategy_guide.md` - Pit stop strategy, overtaking analysis
- `weather_patterns.md` - Historical weather, temperature patterns
- `tire_degradation.md` - Compound performance, stint lengths

## Available Circuits

Documents will be added as races approach or generated from templates.

### Generating Documents

Use the "Generate Templates" button in the RAG sidebar to auto-generate
documents based on historical OpenF1 data.
